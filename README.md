## 📊 Project Overview
This project analyzes sales data from **three coffee shops** to answer key business questions:
- Which products sell the most?
- Which coffee shop performs the best overall?

The analysis combines **SQL queries in Databricks** with **Excel pivot tables, filters, and graphs** to provide clear insights into sales performance.

---

## 🛠 Tools & Technologies
- **Databricks** → Used for SQL coding and data preparation  
- **Excel (CSV export)** → Used for pivot tables, filtering, and visualization  
- **Pivot Tables & Graphs** → To summarize and visualize sales trends  

---

## 🔍 Key Insights
- **Top-Selling Products** → Identified by aggregating transaction quantities and revenue.  
- **Best Performing Coffee Shop** → Determined by comparing total sales and revenue across locations.  
- **Sales Trends** → Visualized with graphs to highlight seasonal or daily performance differences.

## 🚀 How to Use
1. Run the SQL queries in **Databricks** to clean and prepare the dataset.  
2. Export the results as a **CSV file**.  
3. Open the CSV in **Excel** and apply pivot tables, filters, and graphs.  
4. Review the insights on product performance and shop comparisons.
